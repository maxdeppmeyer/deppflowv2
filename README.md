# Deppmeyer Workflow Solutions – V2 Premium Website

Diese Version ist eine komplette visuelle Neugestaltung der bestehenden statischen Webseite.

## Enthalten

- Responsive Premium-Design für Desktop, Tablet und Handy
- Dark/Light Theme
- Scroll-Reveal-Animationen
- 3D-Tilt-Effekte auf Karten und Visuals
- Schnellcheck mit Übergabe ins Kontaktformular
- Kontaktformular bleibt mit dem vorhandenen Cloudflare Worker kompatibel
- Zentrale Unternehmensdaten in `site-config.js`

## Vor dem Livegang prüfen

1. `site-config.js` mit echten Kontaktdaten ausfüllen.
2. `impressum.html` und `datenschutz.html` rechtlich prüfen und final ergänzen.
3. In Cloudflare die Worker-/Pages-Variablen für Kontaktformular und n8n setzen.
